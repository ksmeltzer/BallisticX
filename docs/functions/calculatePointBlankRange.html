      Description

      Computes point-blank-range related metrics for a projectile via the Runge-Kutta Methods (RK4):

      • Notes on stability:
      • Adaptive timestep is used (dt = 0.5 / v) but we cap dt to avoid very large timesteps when velocity becomes small (which would otherwise cause runaway vertical velocities).
      • If the integrator becomes numerically unstable (v ≈ 0 or vertical velocity >> horizontal), we log and return the current (zeroed) result rather than throwing an uncaught error.
